Web Components

Como Usar Web Components

Inclua o componente utilizando import:

	
	<link rel="import" href="meu-componente.html">

Como Usar Web Components

Ou inclua o componente como JS:

	
		<script src="meu-components.js">

Como Usar Web Components

Use como uma tag normal:

	
		<meu-componente>Oi</meu-componente>

Como Usar Web Components

	
		<gaia-header action="back">
		  <h1>Keyboard Settings</h1>
		</gaia-header>

Como Usar Web Components

	
		<gaia-header">
		<h1>really long title</h1>
		<a data-icon="email"></a>
		<button data-icon="settings"></button>
		</gaia-header>
